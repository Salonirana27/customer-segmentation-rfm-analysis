# customer-segmentation-rfm-analysis
Customer Segmentation and Behaviour Analysis using Python (Pandas, NumPy, Matplotlib) with 1M(10lakh) dataset

## 📌 Project Overview
This project analyzes customer behavior using RFM (Recency, Frequency, Monetary) analysis.

## 📊 Dataset
- 1 Million synthetic customer transaction records
- Generated using Python (NumPy & Pandas)

## ⚙️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 🔍 Methodology
- Data generation
- Data preprocessing
- RFM calculation
- Customer segmentation
- Data visualization

## 👥 Customer Segments
- VIP
- Loyal
- New
- At Risk
- Lost

## 📈 Visualizations
- Bar Chart (Segmentation)
- Pie Chart (Distribution)
- Top Customers
- Revenue Contribution

## 💡 Key Insights
- VIP customers generate highest revenue
- Loyal customers provide consistent sales
- At Risk customers need retention strategies
- Lost customers require re-engagement

## 🚀 How to Run
1. Run `generate_dataset.py`
2. Open `analysis.ipynb`
3. Execute all cells
